Welcome to Flask UI

This is a web application that allows you to access data from a .ttl file using key words or natural language.

A .ttl file is a text file that contains data in the Turtle format, which is a syntax for expressing structured information using the Resource Description Framework (RDF). RDF is a standard model for data interchange on the Web.

To use this application, you need to enter a query in the textarea below. The query can be either a key word or a natural language sentence. A key word is a single word or a phrase that represents a concept or an entity in the data. A natural language sentence is a grammatically correct sentence that expresses a question or a request about the data.

For example, if you want to find out the name of the crops grown in Kenya, you can enter either “crops in Kenya” as a key word or “Which crops have the highest yield in Kenya in 2010?” as a natural language sentence. The application will search the .ttl file for the relevant data and return the answer to you.


Sample Queries:

  • Crops in Kenya
  • Which crop have the highest yield in Kenya in 2010?
  • How does the average temperature affect the yield of Wheat in Angola?
  • Which countries use more than 50 units of pesticide on Wheat in 2010?
  • What is the Maize production in Kenya in 2013?
